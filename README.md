# AI POWERED CUSTOMER SUPPORT CHATBOT
An AI-powered customer support chatbot platform that allows businesses to deploy intelligent assistants on their websites using a simple embed script.

Authentication is powered by Scalekit, enabling secure multi-tenant access and dashboard management.

## Features
✨Features
🔐 Secure authentication (Scalekit)
🧠 AI-powered chatbot responses
🏢 Multi-tenant support (Owner-based chatbot instances)
📊 Dashboard for chatbot management
🌐 Easy website embed via script tag
⚡ Real-time chat experience
🎨 Modern UI built with React + Tailwind
🎥 Smooth animations using Framer Motion
